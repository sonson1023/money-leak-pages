# 새는돈 (Money Leaks) 개인정보처리방침

**최종 수정일**: 2026년 7월 5일
**앱 이름**: 새는돈 (Money Leaks)
**패키지명**: `com.wanhee.money_leaks`
**연락처**: 앱 내 [설정] → [앱 정보] 또는 개발자 이메일

---

## 1. 개요

「새는돈」은 구독과 고정 지출을 한눈에 보고, 새는 돈을 찾아 해지 체크리스트로 정리해 드리는 가계 보조 앱입니다.

본 앱은 **사용자가 입력하는 모든 데이터를 사용자의 기기 내부에만 저장**하며, 외부 서버로 전송하거나 백업하지 않습니다. 본 개인정보처리방침은 본 앱이 어떤 정보를 수집·이용하는지, 그리고 사용자의 권리가 무엇인지 설명합니다.

## 2. 수집하는 정보

### 2-1. 사용자가 직접 입력하는 데이터 (기기에만 저장)

본 앱은 서비스 제공을 위해 아래 데이터를 사용자의 기기 내부 저장공간(Hive 로컬 데이터베이스)에 저장합니다.

| 항목 | 예시 | 용도 |
|---|---|---|
| 구독/지출명 | "넷플릭스", "유튜브 프리미엄" | 목록 표시·관리 |
| 금액 | 월 14,500원 | 고정비 합산·통계 |
| 결제 주기/결제일 | 월간/연간, 매월 15일 | 다음 결제일 산출·알림 |
| 카테고리 | OTT, 음악, 클라우드 등 | 도넛 차트 통계 |
| 시작일, 메모, 이모지 | — | 사용자 편의 |
| 방치 추천용 사용일 | (선택 입력) | 90일+ 미사용 구독 추천 |

> **이 데이터는 사용자의 기기에서만 보관되며, 개발자나 제3자가 열람할 수 없습니다.** 계정·로그인이 필요 없습니다.

### 2-2. 자동으로 수집되는 정보

본 앱 자체는 사용자의 개인정보를 자동 수집하지 않습니다. 단, 광고 표시를 위해 타사(AdMob) SDK가 아래 정보를 처리할 수 있습니다.

- 기기 식별자, IP 주소, 운영체제 정보, 대략적 위치 정보
- 자세한 내용은 제4항(광고 및 제3자 서비스)을 참고하세요.

## 3. 데이터의 저장 및 보관

- **저장 위치**: 기기 내부 저장공간 (`getApplicationDocumentsDirectory()`)
- **저장 방식**: Hive 로컬 데이터베이스 (앱 전용 비암호화 로컬 저장)
- **보관 기간**: 사용자가 삭제할 때까지. 앱을 삭제하면 모든 데이터도 함께 삭제됩니다.
- **외부 서버 저장**: 없음 (서버 전송, 클라우드 동기화 미지원)

### 백업/복원 기능 (사용자 주도)

본 앱은 사용자가 직접 JSON 파일로 데이터를 내보내거나 가져올 수 있는 기능을 제공합니다.
- 이 기능은 **사용자가 명시적으로 실행할 때만** 동작합니다.
- 내보낸 파일은 기기 내 문서 폴더에 저장되며, 파일의 보관과 이동은 **사용자의 책임**입니다.
- 파일에 민감 정보가 포함될 수 있으니 안전하게 관리해 주세요.

## 4. 광고 및 제3자 서비스

본 앱은 무료 서비스 운영을 위해 **Google AdMob** 배너 광고를 표시합니다.

- **광고 제공자**: Google LLC
- **위치**: 앱 하단 배너 영역
- **광고 식별자**: 기기의 광고 ID(Google Play 서비스)가 광고 타겟팅·빈도 제한·사기 방지에 사용될 수 있습니다.
- **데이터 처리**: AdMob SDK가 기기 식별자·IP 주소·운영체제 버전·대략적 위치 등을 수집하여 맞춤 광고에 활용할 수 있습니다.
- **Google 개인정보처리방침**: https://policies.google.com/privacy
- **AdMob 개인정보처리방침**: https://support.google.com/admob/answer/6128543

> 사용자는 기기 설정에서 광고 ID 재설정 또는 맞춤 광고 opt-out을 할 수 있습니다.
> (Android: 설정 → Google → 광고)

### 기타 제3자 서비스

본 앱은 AdMob 외에 분석(Firebase Analytics 등), 크래시 리포트(Sentry/Crashlytics), 원격 서버 API, 소셜 로그인 등 **어떠한 제3자 서비스도 사용하지 않습니다.**

## 5. 데이터의 이용 목적

수집/저장된 데이터는 다음 목적에만 사용됩니다.

1. 구독/고정비 목록 표시·관리
2. 월간·연간 고정비 통계 및 도넛 차트 제공
3. 결제일 D-day 알림(기기 로컬 알림)
4. 방치 추천(사용 빈도 기반 해지 권장)
5. 무료 서비스 운영을 위한 광고 표시 (AdMob)

## 6. 사용자 권리

사용자는 언제든지 본인의 데이터를 통제할 수 있습니다.

- **열람·수정**: 앱 내에서 구독 정보를 자유롭게 조회·편집할 수 있습니다.
- **삭제**: 개별 구독 삭제 또는 앱 삭제 시 모든 데이터가 즉시 삭제됩니다.
- **내보내기**: [설정] → [백업]에서 JSON 파일로 내보낼 수 있습니다.
- **맞춤 광고 거부**: 기기 설정 → Google → 광고에서 광고 ID 재설정/opt-out.

## 7. 데이터 보안

본 앱은 데이터를 기기에만 저장하므로 네트워크를 통한 유출 경로가 없습니다. 다만, 기기 자체의 분실·루팅·악성코드 등에 대한 보안은 사용자의 기기 보호 책임입니다. 백업 파일은 민감 정보를 포함할 수 있으므로 안전하게 보관해 주세요.

## 8. 아동의 프라이버시

본 앱은 13세 미만 아동을 대상으로 하지 않으며, 13세 미만 아동으로부터 고의로 개인정보를 수집하지 않습니다. 보호자가 아동의 개인정보가 수집된 것으로 판단될 경우 연락 주시면 즉시 조치하겠습니다.

## 9. 국외 이전

사용자 입력 데이터는 기기에만 저장되어 국외로 이전되지 않습니다. 단, 광고(AdMob)를 통해 처리되는 데이터는 Google의 글로벌 인프라에서 처리될 수 있습니다. 자세한 내용은 [Google 개인정보처리방침]을 참고하세요.

## 10. 개인정보처리방침 변경
